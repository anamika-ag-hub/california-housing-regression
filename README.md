# California Housing Regression

## Objective
To predict house prices using regression techniques.

## Dataset
California Housing dataset from sklearn.

## Models Used
- Linear Regression
- Decision Tree Regressor

## Steps Performed
- Loaded the dataset
- Converted into DataFrame
- Checked for missing values
- Applied feature scaling using StandardScaler
- Trained models
- Evaluated using MSE, MAE, and R² score

## Conclusion
Decision Tree performed better than Linear Regression with lower error values and higher R² score.
